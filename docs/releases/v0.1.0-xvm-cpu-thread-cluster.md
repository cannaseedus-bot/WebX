# v0.1.0-xvm-cpu-thread-cluster — Audit

**Status:** Fully audited. Merge branch: `merge/xvm-cpu-cluster`

## XVM Core
- 32-fiber cluster VM, phase ∈ [0,6), per-fiber: pc/sp/phase/flags/entropy/pressure/r0-r3
- Cluster barrier: all 32 fibers must reach same phase before release
- Manifold opcodes: OP_GEODESIC(0x40), OP_ENTROPY_GRADIENT(0x41), OP_RIEMANN_CURVATURE(0x42), OP_FOLD_ENTER(0x43), OP_FOLD_EXIT(0x44), OP_PRESSURE_PROPAGATE(0x45)

## SCX Runtime
- KBC1 binary format (magic + version + symbol table + bytecode)
- SCXG geometry streaming (SVG/XML visualization)
- SCXQ2 v1.2: INT4 nibble-packed, per-group dequant scales, BSON metadata, DDS chunks
- `run_train_from_shard()`: on-the-fly context windows from INT8 token tiles

## SMGM-16 (smgm16.py)
- 52 CardSlot modules: field_u[16×192], field_s[192], field_v[192×768] (SVD-like decomp)
- Per-card geometric scalars: amplitude[4], gradient[4], curvature[1], pi_mod[1], adjacency_strength[8]
- Maya embedding (15→192→768): topological feature encoding
- Phase embedding (1→96→768): phase-aware computation
- Multi-loss: task(1.0) + balance(0.1) + stage_balance(0.05) + entropy(0.02)
- 15 engineered token features (fractions, sqrt, sin/cos, mod 3/5/7, phase-modulo)
